\section*{問題}

% 問題文を挿入（変数で置換）
次の方程式 $x^2 - 5x + 6 = 0$ の解として正しいものを選びなさい。

% 選択肢を挿入
\begin{description}[labelwidth=1cm, leftmargin=!]
\choice{A}{$x = 1, x = 6$}
\choice{B}{$x = 2, x = 3$}
\choice{C}{$x = -2, x = -3$}
\choice{D}{$x = 1, x = -6$}
\end{description}

% 解答セクション（オプショナル - 表示したくない場合はコメントアウト）
\section*{解答}

% 解答を挿入（変数で置換）
正解は B) $x = 2, x = 3$ です。

% 解説セクション（オプショナル - 表示したくない場合はコメントアウト）
\section*{解説}

% 解説を挿入（変数で置換）
$x^2 - 5x + 6 = 0$ を因数分解すると $(x - 2)(x - 3) = 0$ となる。
したがって、$x = 2$ または $x = 3$ が解である。
